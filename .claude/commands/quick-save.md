# ⚡ 빠른 저장

급하게 자리를 비워야 할 때 현재 작업을 빠르게 저장합니다.

## WIP (Work In Progress) 커밋

작업 중인 코드를 임시로 저장합니다.

실행할 명령:
```bash
git add .
git commit -m "WIP: 작업 중 임시 저장"
git push
```

## 특징

- **커밋 메시지**: "WIP"로 시작 (작업 중임을 표시)
- **빠른 실행**: 즉시 GitHub에 업로드
- **나중에 정리**: 다음에 이어서 작업 후 정식 커밋으로 변경 가능

## 다음 PC에서 이어서 작업하기

1. `/sync-from-github` - 최신 코드 가져오기
2. 작업 계속하기
3. 완료 후 정식 커밋으로 수정
