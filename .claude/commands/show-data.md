# 📊 최근 수집 데이터 확인

데이터베이스에서 최근 수집된 데이터를 확인하겠습니다.

## 확인할 내용

1. **뉴스 데이터**: 최근 수집된 뉴스 개수와 제목
2. **신용평가 데이터**: 최근 신용평가 보고서
3. **KDI 데이터**: 최근 정책 자료

- 중복 데이터 여부
